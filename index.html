🔑 authentication guide

🎫 personal access token method

  1. 🌐 navigate to github.com/settings/tokens
    2. Opens GitHub token settings in new tab
  2. ➕ click "generate new token (classic)"
  3. ✅ select the "repo" scope checkbox
  4. 📋 copy the generated token to clipboard
  5. 🔐 use this token as your password when prompted

🔐 ssh key authentication

  1. 🔑 ssh-keygen -t ed25519 -C "your-email@example.com"
  2. 🔧 ssh-add ~/.ssh/id_ed25519
  3. 📋 cat ~/.ssh/id_ed25519.pub
  4. 🌐 add the public key to github ssh settings
    5. Opens GitHub SSH key settings in new tab
  5. 🔄 git remote set-url origin git@github.com:user/repo.git

⚡ quick configuration setup

git config --global user.name "your-username" git config --global user.email "your-email@example.com" git push origin main # 🔐 use your personal access token as password
