Workflow Configuration (YAML)

GitHub Authentication Setup

üîë Option 1: Personal Access Token (Recommended)

1. Go to GitHub Settings ‚Üí Tokens

2. Click "Generate new token (classic)"

3. Select repo scope for full repository access

4. Copy the generated token

5. Use token as password when Git prompts

üîê Option 2: SSH Key Setup

1. Generate SSH key: ssh-keygen -t ed25519 -C "your_email@example.com"

2. Add to SSH agent: ssh-add ~/.ssh/id_ed25519

3. Copy public key: cat ~/.ssh/id_ed25519.pub

4. Add to GitHub SSH Keys

5. Change remote URL: git remote set-url origin git@github.com:username/repo.git

üí° Quick Fix Commands

git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"
git push origin main # Will prompt for username/token
