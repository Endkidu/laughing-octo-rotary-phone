🔑 Auth

🎫 Token

  1. 🌐 github.com/settings/tokens
  2. ➕ Generate new
  3. ✅ Select repo
  4. 📋 Copy token
  5. 🔐 Use as password

🔐 SSH

  1. 🔑 ssh-keygen -t ed25519
  2. 🔧 ssh-add ~/.ssh/id_ed25519
  3. 📋 cat ~/.ssh/id_ed25519.pub
  4. 🌐 Add to GitHub
  5. 🔄 git remote set-url origin git@github.com:user/repo.git

💀 Quick

git config --global user.name "username"
git config --global user.email "email@domain.com"
git push origin main # 🤖 Use token as password
